# New Project Agent Adaptive

## Step 1. Analyze the user's image

If the user's image is phone-only or in portrait orientation, ask the user if they want their layout
to work on foldables and tablets. If they say use, CONTINUE.

If the user's image is landscape or they ask for a foldable or tablet design, make sure that the
layout looks good on a phone or in portrait mode. Use a feed, list-and-detail, or supporting pane
layout to make the UI adaptive
